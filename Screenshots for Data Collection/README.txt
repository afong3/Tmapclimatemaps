Found this shapefile of Napa County after digging for a while. 
Usually it is easy to find county/city boundaries but the Napa government website was down as I was trying to find it.
Had to sift through websites - Berkeley conveniently had a shapefile in multiple formats to download the file. 
https://geodata.lib.berkeley.edu/catalog/ark28722-s7x88m

This file is in the EPSG:4326 (WGS84) projection. Can also be downloaded in KMZ or GeoJSON.
